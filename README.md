## Overview

Qengine (3D) extends the 2D quansistor simulator into a **three-dimensional volumetric substrate**.
The goal is not realism, but exploration of how operator-based locality, causality, and auditability scale with dimensional complexity.

---

## What Changes in 3D

- Volumetric neighborhoods
- Surface-based region boundaries
- Richer connectivity and topology
- Increased partitioning complexity

What **does not change**:
- operator-first semantics,
- no-signalling guarantees,
- deterministic replay,
- invariant-based auditability.

---

## Key Contributions

- Defines 3D quansistor lattices
- Generalizes invariant fields volumetrically
- Extends mobile composite objects to 3D
- Preserves causal scheduling and determinism
- Demonstrates scalable audit under complexity

---

## Architectural Insights

Qengine (3D) shows that:
- complexity increases cost, not semantics,
- causality remains enforceable,
- invariants remain sufficient for verification,
- global coordination is unnecessary.
